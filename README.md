# tarea-semana-1
Preguntas - Semana 1

Pregunta 1:
¿Cuál de las siguientes opciones describe mejor qué es la ciudadanía digital?

Respuesta: B. El uso responsable de la tecnología para informarse, participar y compartir información de manera ética.

Pregunta 2:
¿Cuál es la razón principal por la que es importante verificar la información antes de compartirla en internet?

Respuesta: C. Para evitar difundir información falsa o engañosa.

Pregunta 3:
¿Cuál de las siguientes opciones define mejor un algoritmo?

Respuesta: B. Una lista de pasos ordenados para resolver un problema.
 Reflexión de la semana.
 La relación entre el uso responsable de la información en internet, la participación ciudadana y herramientas como Git es que todas requieren responsabilidad y organización. En internet debemos verificar la información antes de compartirla para evitar confusiones o noticias falsas. La participación ciudadana permite que las personas expresen sus ideas y colaboren en diferentes espacios digitales. Herramientas como Git ayudan a organizar el trabajo técnico, guardar cambios y trabajar en equipo. Usar estas tecnologías de forma correcta fortalece la confianza entre las personas. También permite crear proyectos más seguros y ordenados. Por eso es importante tener buenos hábitos digitales y aprender a utilizar la tecnología responsablemente.
